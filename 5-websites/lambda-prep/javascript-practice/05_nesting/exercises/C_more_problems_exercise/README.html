Instructions

Create a new folder named C_more_problems, this is where you'll save all of your files. Solve each of the prompts below. Create a new file for each prompt and execute your code to see if it matches the example output.

This problem set tests a few different concepts. Any concepts we've covered are fair game. Some of the following problems are tough, but they only require knowledge of concepts we've already covered. Give them a shot.

removeDupes

Create a remove-dupes.js file and solve the function below:

// Write a function `removeDupes` that accepts
                  an array as an argument. The function should return a
              //
                  new array where each element only appears once.
              
              console.log(removeDupes(["x", "y", "y", "x", "z"])); // ['x', 'y', 'z']
              console.log(removeDupes([false, false, true, false])); // [false, true]
              console.log(removeDupes([42, 5, 7, 42, 7, 3, 7, 7]));
                // [42, 5, 7, 3]

spam

Create a spam.js file and solve the function below:

// Write a function `spam` that accepts a 2D
                  array as an argument. The array contains pairs as elements.
              //
                  The first element of every pair is a number and the second element is a word. The function
                  should
              //
                  return a string containing the words repeated the specified number of times. See the
                  examples.
              
              let array1 = [
               ["hi", 3],
               ["bye", 2],
              ];
              console.log(spam(array1)); // 'hi hi hi bye bye'
              
              let array2 = [
               ["cat", 1],
               ["dog", 2],
               ["bird", 4],
              ];
              console.log(spam(array2)); // 'cat dog dog bird bird bird bird'
