Data Types and Variables

Data Types and Variables

Data Types

In the last unit, we discussed how expressions are values combined with operators, resulting in a single value. The category for these values are data types. In simple words, it's the type of data we use. Every value belongs to exactly one data type.

Common Data Types:

Data Type Example
Integers -3, -2, -1, 0, 1, 2, 3
Floating point numbers -1.5, 2.0, 5.75
Strings 'student', 'Hello!', '11'

For Strings, we need to use '' to remind python that the string has ended. Otherwise, we face the following error: 

>>> 'Hello world!
SyntaxError: EOL while scanning string literal

Since we forgot to close the String with a single quote, we faced the error.

More on Strings

The first question that comes to mind is, what happens when we use a + operator with Strings. Lets try it out: 

>>> 'hello'+'hello'
'hellohello'

This is referred to as String Concatenation. The two strings are joined together while + acts as a string concatenation operator.

However, note one thing. We can't use + to add two different data types. 

>>> 'Student' + 42
Traceback (most recent call last):
  File "<python>", line 1, in <module>
    'Student' + 42
TypeError: Can't convert 'int' object to str implicitly

The error message Can't convert 'int' object to str implicitly means that Python thought you were trying to concatenate an integer to the string 'Alice'. Your code will have to explicitly convert the integer to a string, because Python cannot do this automatically. We will discuss this further in the coming units.

What if we use * operator on Strings? What happens then? Lets find out: 

>>>'Student'*5
'StudentStudentStudentStudentStudent'

This is referred to as String replication. However, remember that it's not possible to multiply a string with another string or a floating point number. 

>>> 'A' * 'B'
Traceback (most recent call last):
  File "<python>", line 1, in <module>
    'A' * 'B'
TypeError: can't multiply sequence by non-int of type 'str'

>>> 'A' * 5.0
Traceback (most recent call last):
  File "<python>", line 1, in <module>
    'A' * 5.0
TypeError: can't multiply sequence by non-int of type 'float'

It makes sense that Python wouldn’t understand these expressions: You can’t multiply two words, and it’s hard to replicate an arbitrary string a fractional number of times.

These are some useful tricks while working on Python programs!

Variables

A variable can be called a box in the Computer's memory, that can store exactly one value. This has several important use cases. Lets say we are running a program and wish to save the value of an evaluated expression, for later use. We can do this with a variable.

Example:

>>> a = 5 + 7
>>> a
12

Here we see that a is assigned the value of 5 + 7 and stores the value 12. The value is printed when we call a again.

We will discuss this further in the next unit!
